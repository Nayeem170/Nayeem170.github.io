# CV Website - Nayeem Bin Ahsan

A responsive CV website built with HTML, CSS, and jQuery.

## Features

- Embedded data for local operation without a server
- Responsive design (desktop, tablet, mobile)
- Modern UI with gradient accents and smooth animations
- No build process required

## Quick Start

Simply open `index.html` in any modern web browser. No server setup needed!

## Customization

All CV content is in `script.js` (the `cvData` object). Modify the sections as needed and refresh the page.

To change colors, edit the CSS variables in `styles.css` (primary colors: `#667eea`, `#764ba2`).

## Deploy to GitHub Pages

1. Push your code to GitHub repository
2. Go to repository **Settings** → **Pages**
3. Under **Build and deployment** → **Source**, select **GitHub Actions**
4. The workflow will automatically deploy on push to `main` branch
5. Your site will be available at: `https://username.github.io/repository-name`

## Technologies

- HTML5
- CSS3 (Flexbox, Grid)
- jQuery 3.7.1
- Font Awesome 6.4.0
